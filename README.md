📌 PDV API - Sistema de Vendas

API REST de um sistema de ponto de venda (PDV), desenvolvida com Java e Spring Boot.

🚀 Funcionalidades
✅ Cadastro de produtos
✅ Controle de estoque
✅ Criação de vendas
✅ Adição de itens na venda
✅ Finalização de venda
✅ Pagamento com cálculo de troco
🛠️ Tecnologias
Java 17+
Spring Boot
Spring Data JPA
Hibernate
Maven
H2 / MySQL
